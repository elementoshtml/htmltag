HTML:

Elemento raíz de un documento HTML. Es el contenedor principal que engloba todo el contenido de la página.

<html>
<head></head>
<body></body>
</html>

HEAD:

Contiene metadatos sobre el documento, como el título de la página y enlaces a archivos externos como hojas de estilo.

<head>
<title>Título de la página</title>
</head>

BODY:

Contiene el contenido visible de la página web, como textos, imágenes, y otros elementos que los usuarios ven en el navegador.

<body>
<p>Contenido visible aquí</p>
</body>

TITLE:

Define el título del documento, el cual aparece en la pestaña del navegador.

<title>Mi Página</title>

SCRIPT:

Define un script de JavaScript que se puede ejecutar en la página.

<script>
console.log('Hola, mundo');
</script>

STYLE:

Define estilos CSS internos para aplicar directamente a los elementos del documento.

<style>
p { color: blue; }
</style>

ADDRESS:

Proporciona información de contacto del autor o de la página.

<address>
Contacto: ejemplo@example.com
</address>

P:

Define un párrafo de texto.

<p>Este es un párrafo de texto.</p>

BR:

Inserta un salto de línea, útil para crear espacios en el texto.

<p>Línea 1<br>Línea 2</p>

PRE:

Muestra texto preformateado, respetando espacios y saltos de línea.

<pre>
Texto preformateado
con espacios y saltos de línea.
</pre>

BLOCKQUOTE:

Indica una cita en bloque, separando el texto citado del resto del contenido.

<blockquote>
Cita en bloque.
</blockquote>

OL:

Crea una lista ordenada, donde cada elemento tiene un número.

<ol>
<li>Elemento 1</li>
<li>Elemento 2</li>
</ol>

UL:

Crea una lista desordenada, donde cada elemento tiene un punto.

<ul>
<li>Elemento A</li>
<li>Elemento B</li>
</ul>

LI:

Define un elemento de una lista.

<ul>
<li>Elemento de lista</li>
</ul>

DL:

Inicia una lista de definiciones, donde cada término se define con una descripción.

<dl>
<dt>Término</dt>
<dd>Definición del término</dd>
</dl>

DT:

Define un término dentro de una lista de definiciones.

<dl>
<dt>Término</dt>
</dl>

DD:

Proporciona la definición del término en una lista de definiciones.

<dl>
<dt>Término</dt>
<dd>Definición</dd>
</dl>

FIGURE:

Agrupa contenido ilustrativo, como imágenes, junto con su pie de foto.

<figure>
<img src="imagen.jpg" alt="Descripción de la imagen">
<figcaption>Pie de foto</figcaption>
</figure>

FIGCAPTION:

Proporciona un pie de foto para el elemento <figure>.

<figure>
<img src="imagen.jpg" alt="Descripción de la imagen">
<figcaption>Pie de foto</figcaption>
</figure>

DIALOG:

Representa un cuadro de diálogo o ventana emergente que puede ser mostrado o escondido.

<dialog open>
<p>Este es un cuadro de diálogo.</p>
<button onclick="this.closest('dialog').close()">Cerrar</button>
</dialog>

A:

Define un enlace a otra página o recurso.

<a href="pagina.html">Enlace</a>

EM:

Indica énfasis en el texto, generalmente se muestra en cursiva.

<p>Texto <em>enfatizado</em>.</p>

STRONG:

Indica que el texto es importante, usualmente se muestra en negrita.

<p>Texto <strong>importante</strong>.</p>

SMALL:

Reduce el tamaño del texto.

<p>Texto <small>pequeño</small>.</p>

CITE:

Se utiliza para citar una fuente de información.

<p>Según <cite>El Gran Libro</cite>.</p>

S:

Representa texto que ha sido eliminado o que ya no es relevante.

<p>Texto <s>eliminado</s>.</p>

Q:

Indica una cita en línea, generalmente con comillas alrededor.

<p>Ella dijo <q>esto es una cita</q>.</p>

DFN:

Define un término en el contexto de una definición.

<p>Un <dfn>glosario</dfn> es una colección de términos.</p>

ABBR:

Representa una abreviatura o acrónimo con su descripción completa.

<p>La <abbr title="World Health Organization">WHO</abbr> es una organización internacional.</p>

DATA:

Asocia contenido con un valor específico para facilitar el análisis.

<data value="10">Diez</data>

TIME:

Representa una fecha y hora, útil para información temporal.

<time datetime="2024-07-29">29 de julio de 2024</time>

CODE:

Define un fragmento de código, generalmente utilizado para mostrar código fuente.

<p>El código para
"Hola Mundo" es <code>console.log('Hola Mundo')</code>.</p>

VAR:

Representa una variable en un contexto de programación.

<p>La variable <var>x</var> se usa para almacenar valores.</p>

SAMP:

Muestra un resultado de ejemplo de un programa o código.

<p>El resultado de la ejecución es <samp>42</samp>.</p>

KBD:

Indica texto que se debe escribir en un teclado.

<p>Presione <kbd>Ctrl</kbd> + <kbd>S</kbd> para guardar.</p>

SUB:

Define texto subíndice, que se muestra más pequeño y más bajo que el texto normal.

<p>Texto con subíndice <sub>2</sub>.</p>

SUP:

Define texto superíndice, que se muestra más pequeño y más alto que el texto normal.

<p>Texto con superíndice <sup>2</sup>.</p>

I:

Representa texto en cursiva.

<p>Texto en <i>cursiva</i>.</p>

B:

Representa texto en negrita.

<p>Texto en <b>negrita</b>.</p>

MARK:

Resalta texto para llamar la atención.

<p>Texto <mark>resaltado</mark>.</p>

RUBY:

Define anotaciones para caracteres de ideogramas, usado en textos asiáticos.

<ruby>漢字<rt>kanji</rt></ruby>

RB:

Define una parte de la anotación ruby.

<ruby>漢字<rb>kanji</rb></ruby>

RT:

Define la pronunciación o lectura de un carácter en ruby.

<ruby>漢字<rt>kanji</rt></ruby>

RTC:

Define un grupo de caracteres ruby para su anotación.

<ruby>漢字<rtc>kanji</rtc></ruby>

RP:

Define un elemento opcional para la anotación ruby, como corchetes.

<ruby>漢字<rp>(</rp><rt>kanji</rt><rp>)</rp></ruby>

BDO:

Cambia la dirección del texto, útil para lenguajes que se escriben de derecha a izquierda.

<p><bdo dir="rtl">Texto en RTL</bdo></p>

BDI:

Aísla una parte del texto para que su dirección no afecte al resto del contenido.

<p>Texto normal <bdi dir="ltr">Texto aislado</bdi> texto normal</p>

IFRAME:

Permite incrustar otra página web dentro de la página actual.

<iframe src="pagina.html" width="600" height="400"></iframe>

FORM:

Define un formulario para la entrada de datos del usuario.

<form action="enviar.php">
<input type="text" name="nombre">
<input type="submit" value="Enviar">
</form>

INPUT:

Define un campo de entrada en un formulario.

<input type="text" name="nombre">

TEXTAREA:

Define un área de texto en un formulario.

<textarea name="comentarios">Texto aquí</textarea>

SELECT:

Define un menú desplegable en un formulario.

<select name="opciones">
<option value="1">Opción 1</option>
<option value="2">Opción 2</option>
</select>

BUTTON:

Define un botón que puede ser utilizado para enviar un formulario o ejecutar una acción.

<button type="button">Haz clic aquí</button>

LABEL:

Define una etiqueta para un elemento de formulario.

<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">

FIELDSET:

Agrupa controles relacionados en un formulario.

<fieldset>
<legend>Información personal</legend>
<input type="text" name="nombre">
</fieldset>

LEGEND:

Proporciona un título para el contenido de un <fieldset>.

<fieldset>
<legend>Información personal</legend>
<input type="text" name="nombre">
</fieldset>

OPTGROUP:

Agrupa opciones relacionadas en un menú desplegable.

<select name="opciones">
<optgroup label="Grupo 1">
<option value="1">Opción 1</option>
<option value="2">Opción 2</option>
</optgroup>
<optgroup label="Grupo 2">
<option value="3">Opción 3</option>
</optgroup>
</select>

OUTPUT:

Define un área donde se mostrará el resultado de un cálculo o script.

<output>Resultado del cálculo</output>

DATALIST:

Define una lista de opciones predefinidas para un campo de entrada.

<input list="colores">
<datalist id="colores">
<option value="Rojo">
<option value="Verde">
<option value="Azul">
</datalist>

METER:

Representa una medida escalar dentro de un rango conocido.

<meter value="0.6" min="0" max="1">60%</meter>

PROGRESS:

Muestra el progreso de una tarea en curso.

<progress value="30" max="100">30%</progress>

DETAILS:

Define un control que permite mostrar u ocultar información adicional.

<details>
<summary>Título</summary>
Contenido adicional</details>

SUMMARY:

Proporciona un título o resumen para el contenido dentro de un elemento <details>.

<details>
<summary>Título</summary>
Contenido adicional</details>

SLOT:

Define un espacio dentro de un elemento <template> para contenido dinámico.

<template>
<div>Contenido por defecto</div>
<slot name="mi-slot"></slot>
</template>

TEMPLATE:

Define contenido que no se muestra hasta que se utiliza en otro lugar.

<template>
<div>Contenido de plantilla</div>
</template>

ARTICLE:

Define un contenido independiente y autocontenido.

<article>
<header>Encabezado</header>
<p>Contenido del artículo</p>
</article>

SECTION:

Define una sección temática en un documento.

<section>
<header>Encabezado de la sección</header>
<p>Contenido de la sección</p>
</section>

ASIDE:

Define un contenido que está relacionado de manera indirecta con el contenido principal.

<aside>
<p>Contenido relacionado</p>
</aside>

NAV:

Define un bloque de navegación.

<nav>
<ul>
<li><a href="pagina1.html">Página 1</a></li>
<li><a href="pagina2.html">Página 2</a></li>
</ul>
</nav>

HEADER:

Define un encabezado para un documento o una sección.

<header>
<h1>Título principal</h1>
<p>Descripción</p>
</header>

FOOTER:

Define un pie de página para un documento o una sección.

<footer>
<p>Texto del pie de página</p>
</footer>

MAIN:

Define el contenido principal de un documento.

<main>
<p>Contenido principal</p>
</main>

H1:

Encabezados de diferentes niveles. <h1> es el más alto y <h6> el más bajo en términos de importancia y tamaño.

<h1>Encabezado principal</h1>
<h2>Encabezado secundario</h2>

H2:

Define el nivel 2 del encabezado.

<h2>Encabezado secundario</h2>

H3:

Define el nivel 3 del encabezado.

<h3>Encabezado terciario</h3>

H4:

Define el nivel 4 del encabezado.

<h4>Encabezado cuaternario</h4>

H5:

Define el nivel 5 del encabezado.

<h5>Encabezado quinario</h5>

H6:

Define el nivel 6 del encabezado.

<h6>Encabezado senario</h6>

HR:

Inserta una línea horizontal para separar contenido.

<hr>

DIV:

Define un contenedor de bloque para agrupar contenido.

<div>Contenido dentro de un contenedor</div>

SPAN:

Define un contenedor en línea para agrupar contenido.

<span>Texto en línea</span>

IMG:

Define una imagen.

<img src="imagen.jpg" alt="Descripción de la imagen">

LINK:

Define un enlace a una hoja de estilos CSS.

<link rel="stylesheet" href="estilos.css">

META:

Define metadatos sobre el documento.

<meta charset="UTF-8">
<meta name="description" content="Descripción de la página">

BASE:

Define una URL base para los enlaces en el documento.

<base href="https://www.ejemplo.com/">

NOSCRIPT:

Define contenido alternativo para navegadores que no soportan JavaScript.

<noscript>JavaScript no está habilitado</noscript>

DOCTYPE:

Declara el tipo de documento y la versión de HTML.

<!DOCTYPE html>
